NAME

meteorFormat  

SYNOPSIS

#include <meteor.h>

int meteorFormat(void);  

DESCRIPTION

This function returns the format used by the meteor, previously set by meteorReset, it is METEOR_COORDS by default.  

RETURN VALUE

A format mask specifying the data in the meteor, see meteorReset for more information.
