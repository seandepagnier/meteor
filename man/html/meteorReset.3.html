NAME

meteorReset  

SYNOPSIS

#include <meteor.h>

void meteorReset(int format);  

DESCRIPTION

This function resets the current meteor removing any previous data. The format argument specifies what data is to be stored per-point in the meteor. A format description is a simple bitwise mask of METEOR_COORDS, METEOR_NORMALS, METEOR_COLORS, and METEOR_TEXCOORDS. If the required format METEOR_COORDS is not specified, it is automatically added.
