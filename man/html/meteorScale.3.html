NAME

meteorMultMatrix meteorRotate meteorTranslate meteorScale  

SYNOPSIS

#include <meteor.h>

void meteorMultMatrix(double m[16]);
void meteorRotate(double angle, double x, double y, double z);
void meteorTranslate(double x, double y, double z);
void meteorScale(double x, double y, double z);  

DESCRIPTION

Apply matrix transformations to the meteor for each point. Normals are rotated without translation. meteorMultMatrix applies a given 4x4 matrix. meteorRotate performs a rotation of angle around the vector <x, y, z>. meteorTranslate translates each point. meteorScale applies scaling.
