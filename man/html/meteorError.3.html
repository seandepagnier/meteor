NAME

meteorMerge  

SYNOPSIS

#include <meteor.h>

const char *meteorError(void)  

DESCRIPTION

This function provides human readable error messages when an error occurs. Calling meteorError will clear the last error.  

RETURN VALUE

If an error occured since the last call to meteorError then an error description string is returned, otherwise NULL is returned;
