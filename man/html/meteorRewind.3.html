NAME

meteorRewind  

SYNOPSIS

#include <meteor.h>

void meteorRewind(void);  

DESCRIPTION

This function rewinds the offsets used by meteorReadPoints and meteorReadTriangles.  

NOTES

meteorReset causes an implicit rewind.
