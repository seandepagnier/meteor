METEORPOINTCOUNT

Updated: 2007-02-25

NAME

meteorPointCount meteorPointCreatedCount meteorTriangleCount meteorTriangleCreatedCount meteorTriangleMergeableCount  

SYNOPSIS

#include <meteor.h>

int meteorPointCount(void);
int meteorPointCreatedCount(void);
int meteorTriangleCount(void);
int meteorTriangleCreatedCount(void);
int meteorTriangleMergableCount(void);
 

RETURN VALUE

meteorPointCount returns the number of points currently in the meteor.

meteorPointCreatedCount returns the total number of points created since the last reset. If points are eliminated by operations, this number does not decrease, (this function is useful for statistics)

meteorTriangleCount returns the number of triangles currently in the meteor.

meteorTriangleCreatedCount returns the total number of triangles created since the last reset.

meteorTriangleMergeableCount returns the number of triangles that can be eliminated by meteorMerge during building.
