METEORFUNC

NAME

meteorFunc meteorNormalFunc meteorTextureFunc meteorColorFunc  

SYNOPSIS

#include <meteor.h>

void meteorFunc(double (*func)(double x, double y, double z));
void meteorNormalFunc(void (*func)(double[3], double[3]));
void meteorColorFunc(void (*func)(double[3], double[3]));
void meteorTextureFunc(void (*func)(double[3], double[3]));  

DESCRIPTION

These functions set the callback functions for building a meteor. meteorFunc is required for building, the other functions are optional. The function provided to meteorFunc should return positive values if a point (specified by x, y, z) is outside the surface, negative values inside the surface, and 0 if the point lies on the surface.

The functions meteorNormalFunc, meteorColorFunc, and meteorTextureFunc specify means of calculating these parameters, if NULL is passed the callback is disabled. The first argument of type double[3] is the destination value, and the second argument also of type double[3] is the source location as an x, y, z position.  

NOTES

These functions are invoked while performing various operations on the meteor to improve the results. If not specified, fallbacks (such as averaging the values for two points in the case of merging) are used instead.
